# 真实数据-仿真曲线校准评估

本文件由 `scripts/calibrate_simulation_to_real_data.py` 生成，用于把真实传播代理曲线与 AnyLogic 场景平均曲线进行归一化形状比较。

## 输入

- 真实曲线：`SCI一区升级规划/真实数据处理/checked_fake_i1_proxy_for_calibration.csv`
- 真实时间列：`t_day`
- 真实数值列：`normalized_I1_proxy`
- 仿真 summary：`/Users/hao/2026/Codex/6月/毕业论文/SCI一区升级规划/实验结果汇总/summary_from_raw.csv`
- 仿真 raw 目录：`/Users/hao/2026/Codex/6月/毕业论文/AnyLogic输出数据`
- 仿真信号：`I1`
- 比较网格点：101

## 解释边界

- 该评估使用归一化时间和归一化传播强度，只比较曲线形状，不证明真实人数规模一致。
- 若真实曲线来自 CHECKED，主要支撑线上传播代理 `I1(t)`，不能直接验证线下传播者 `I2(t)`。
- 该结果应作为校准/外部验证的入口，正式投稿前还需要报告数据版本、许可证、访问日期和失败案例。
