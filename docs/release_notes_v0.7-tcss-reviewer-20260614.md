# Release Notes: v0.7-tcss-reviewer-20260614

This release is a sanitized reviewer-access package for the TCSS-oriented manuscript on coupled online-offline SEIRG rumor-refutation intervention simulations.

## Contents

- AnyLogic model files and model-mode notes.
- Raw AnyLogic simulation exports generated by the project.
- Processed simulation summaries, statistical-test tables, and figure source data.
- Manuscript-supporting figures and tables.
- Analysis scripts for queue construction, run summarisation, statistical tests, sensitivity-design generation, and figure support.
- A lightweight standard-library Python mean-field SEIRG reference script for equation-level inspection.
- Local sensitivity and ablation outputs.
- Morris and Latin-hypercube design queues and protocols, provided only as prepared designs rather than completed global sensitivity results.
- Data dictionary, reproducibility guide, data-availability statement, data-licence notice, and SHA256 manifest.

## Data and Licence Boundary

The MIT License applies only to original scripts and documentation. Third-party datasets and platform-derived materials remain governed by their original source terms and are not relicensed by this repository.

The package does not redistribute CHECKED raw JSON/text records, CHECKED microblog-level normalized records, full OxCGRT compact national raw CSV files, IFCN/Poynter raw database exports, or page snapshots. Included external-data derivatives are limited to aggregate or processed proxy files, source access logs, and source registration metadata.

## Claim Boundary

This package supports model-internal comparison of rumor-refutation intervention in a coupled online-offline SEIRG simulation. It does not provide causal evidence that real-world policy action reduced rumor propagation.
